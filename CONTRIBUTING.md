# Contributing

1. Open an issue for larger changes or security-sensitive behavior.
2. Fork the repository, create a branch, and submit a pull request.
3. Run `pnpm install` and `pnpm run build`; fix any new lint issues from `pnpm run lint`.
4. Keep actions and providers focused on Internet Computer use cases (wallets, canisters, ICRC-1, Launchpad-style registries).

## Security

Do not commit private keys, PEM files, or API secrets. Use environment variables only.
